Contexte


Dans le cadre du BTS SIO, la mise en place d’une veille technologique est imposée.
Suite à mon stage chez Vilogia mon choix s'est porté sur le Java et plus précisément sur l'utilisation du Java dans les jeux vidéo.

Présentation


Java est un langage de programmation orienté objet créé par James Gosling et Patrick Naughton, employés de Sun Microsystems, avec le soutien de Bill Joy (cofondateur de Sun Microsystems en 1982), présenté officiellement le 23 mai 1995 au SunWorld.
La société Sun a été ensuite rachetée en 2009 par la société Oracle qui détient et maintient désormais Java.

Choix de veille


Durant mon stage à Vilogia, j'ai du apprendre le Java. Je me suis donc renseigné sur toutes les utilisations possibles de ce langage.
Et mon choix s'est porté sur l'utilisation du Java dans les jeux vidéo. Le Java m'a toute de suite plu donc mon choix s'est porté directement sur ce langage.
Et j'ai choisi les jeux vidéo parce que je suis passioné depuis toujours et j'aimerai en faire mon métier.
